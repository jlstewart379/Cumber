Overview

This class is the wrapper for the UIAPicker and UIAPickerWheel class. The UIAPicker class allows users to interact with picker controls. UIAPicker is a subclass of UIAElement

def values_for_wheel(index)
  response = search_and_execute_command("wheels()[#{index}].values()")
  response = ResponseHelper.process_string_response(response)

  if response
    response.split(',')
  else
    []
  end

end
